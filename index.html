Create Templates for Anything

templ is a powerful CLI tool that helps you create, manage, and use templates for any type of project. From web applications to C++ programs, streamline your development workflow.

Get Started Documentation

Features

Universal Templates

Create templates for any project type - web apps, mobile apps, desktop applications, or any other development project.

Easy to Use

Simple CLI interface with intuitive commands. Create and use templates with just a few keystrokes.

Customizable

Define variables and placeholders in your templates for maximum flexibility and reusability.

Installation

# Using npm
npm install -g templ

# Using cargo
cargo install templ

# Using homebrew
brew install templ

Quick Start

Create a Template

templ create my-template

Use a Template

templ use my-template new-project

List Templates

templ list
